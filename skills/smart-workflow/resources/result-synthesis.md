# 结果整合详细说明

## 整合维度

### 1. 需求维度 (PRD)

**内容来源**: `product-manager` Agent

**输出格式**:
```markdown
# 产品需求文档 (PRD)

## 功能概述
- 功能目标
- 目标用户
- 使用场景

## 功能列表
### 必须功能
- 功能1: 描述
- 功能2: 描述

### 可选功能
- 功能A: 描述
- 功能B: 描述

## 约束条件
- 性能要求
- 安全要求
- 兼容性要求
```

## 输出方式

### 1. Markdown 报告

**文件名**: `smart-flow-report-{date}.md`

**位置**: 项目根目录或指定目录

### 2. 结构化数据

**格式**: JSON

**用途**: 程序化处理、历史记录查询

### 3. 控制台输出

**格式**: 实时进度 + 最终摘要

## 历史记录管理

### 记录结构

```
smart-flow/
├── history/
│   ├── EXEC-2025-01-10-001/
│   │   ├── requirements.json
│   │   ├── tasks.json
│   │   ├── execution.json
│   │   └── report.md
│   ├── EXEC-2025-01-10-002/
│   └── ...
```
