# 自动循环使用说明

## 🚀 一句话启动

```
/循环 <你的任务>
```

然后回答 3 个问题：
1. 需要反复吗？
2. 最多几次？
3. 何时算完成？

---

## 💡 实际例子

```
你: /循环 修复测试

我: 1. 需要反复尝试吗？(是/否)
   2. 最多尝试几次？
   3. 怎么算完成？

你: 是, 20, 测试全部通过

我: 开始自动循环！
```

---

## 📝 其他命令

| 命令 | 用途 |
|-----|------|
| `/循环 任务` | 交互式，适合新手 |
| `/自动循环 任务 次数 20 完成 完成` | 一行搞定 |
| `/反复 任务` | 更口语化 |

---
