# 产品与设计类 Agent 输出模板

> **适用Agent**: product-manager, ui-ux-designer, architect-review, documentation-architect, api-documenter, impact-analyzer-*, 等（8个）
> **版本**: v2.1.0
> **更新**: 2025-01-07

---

## 模板说明

本模板适用于所有**产品与设计类** Agent，包括：
- 产品经理
- UI/UX 设计师
- 架构审查
- 文档架构师
- 需求分析

## 使用说明

### 在技能中引用模板

```markdown
## Agent 调用指令

调用 `product-manager` agent 时，请遵循以下要求：

1. **输出格式**: 使用 `templates/agent-outputs/product-design-template.md`
2. **章节要求**: 包含所有必需章节
3. **质量标准**: 输出质量评分 ≥90分
```

### 模板验证（宽松模式）

由于采用宽松验证：
- Agent 输出后**不强制检查格式**
- 建议在 Agent 提示中**明确引用模板**
- 输出结果由用户**手动检查**
