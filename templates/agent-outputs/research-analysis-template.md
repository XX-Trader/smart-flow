# Smart Flow v2.1.0 - 研究与分析类 Agent 输出模板

**模板版本**: v2.1.0
**适用 Agent 类型**: 研究与分析类
**创建日期**: 2026-01-07

---

## 📋 模板说明

本模板适用于所有研究与分析类 Agent，涵盖以下 10 个 Agent：

1. **research-orchestrator** - 研究协调器
2. **comprehensive-researcher** - 综合研究员
3. **technical-researcher** - 技术研究员
4. **academic-researcher** - 学术研究员
5. **market-research-analyst** - 市场研究分析师
6. **search-specialist** - 搜索专家
7. **query-clarifier** - 查询澄清专家
8. **research-coordinator** - 研究协调员
9. **research-synthesizer** - 研究综合员
10. **report-generator** - 报告生成器

## 🎨 通用输出格式规范

### 格式标准
1. **Markdown 语法**: 使用标准 Markdown
2. **表格规范**: 对齐工整,表头明确
3. **图表**: 使用 ASCII 图表或 Mermaid
4. **代码块**: 指定语言,添加注释
5. **链接**: 使用描述性链接文本

### 质量标准
1. **准确性**: 数据有来源,结论有依据
2. **完整性**: 覆盖所有关键维度
3. **可读性**: 结构清晰,逻辑顺畅
4. **可操作性**: 建议具体可执行
5. **时效性**: 标注数据时效性

### 评分系统
- **⭐⭐⭐⭐⭐**: 优秀 (90-100分)
- **⭐⭐⭐⭐**: 良好 (80-89分)
- **⭐⭐⭐**: 中等 (70-79分)
- **⭐⭐**: 较差 (60-69分)
- **⭐**: 很差 (<60分)

### 置信度标注
- **高**: 多个独立来源验证,数据一致
- **中**: 部分来源验证,存在不确定性
- **低**: 来源单一或未验证,需谨慎对待
